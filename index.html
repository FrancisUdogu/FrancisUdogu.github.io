Automated visualizations and interactive report using Macros, VBA, Sparklines, Symbols,
Shapes and Custom Formats for a Customer Service Company

In this project, I used different types of Conditional Formatting like Data bars, colour scales and icon sets to highlight specific values in the worksheet. A rank function has also been used to rank the selected agents satisfaction score against the rest of the Customer Service Agents. I also implemented VBA and Macros with triangle buttons to run a code that highlights the highest Customer satisfaction scores between 4.7 and 5 or lowest scores between 0 and 3.
Sparklines and shapes has been added to visually represent Data in this report together with trend based/value based conditional formatting where rules and formulas were applied. In this Service Desk Data, I implemented lookup functions to show both the average monthly calls and the average resolution rate for any selected Customer Service Agent in the drop down menu.
Finally, for the Consistent Annual Growth Rate (CAGR), I applied Custom Number formatting to show positive values as red triangles pointing up, negative values as black triangles pointing down and nothing for 0 values.
