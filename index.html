Creating a Database, Tables and running several queries in them through MYSQL

A Relational Database Management System called MYSQL was used in creating this Database, three different tables and inserting of data into them. I went ahead to join two tables together using JOIN STATEMENTS. I carried out few data cleaning using various string functions like the CONCATENATE, LENGTH, UPPER, LOWER, RTRIM and LTRIM. I also conducted few analysis using some CASE STATEMENTS. Window functions were also used like the OVER PARTITION BY, ROLLING TOTAL, RANKING and DENSE RANKING. Common table expressions - CTEs and Temp Tables were also implemented. After this, I created STORED PROCEDURES inside the Database and also created TRIGGERS that would occur if certain actions are carried out in the Database. Finally, I concluded with creating an EVENT and scheduled it to happen every 30 seconds in the Database.
